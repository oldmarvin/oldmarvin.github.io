centos中安装python3.7

Posted on 2019-05-23

以下记录在阿里云的centos中安装python3.7的过程

# 安装前置环境    
yum install openssl-devel bzip2-devel expat-devel gdbm-devel readline-devel sqlite-devel  mysql-devel gcc gcc-devel python-devel  
yum install libffi-devel -y

# 下载python3.7并安装
wget https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tgz
tar -zxvf Python-3.7.0.tgz
cd Python-3.7.0
./configure --prefix=/usr/local/python3
make&&make install

# 制作软链接
ln -s /usr/local/python3/bin/python3 /usr/bin/python3
ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3

# 安装虚拟环境
pip3 install virtualenv
ln -s /usr/local/python3/bin/virtualenv /usr/bin/virtualenv
virutalenv -p /usr/bin/python3

一个知识点：如何检查centos的安装源？

1）检查pip.conf文件是否存在

cd ~
mkdir .pip
ls ~/.pip

2）直接编辑pip.conf

sudo vi ~/.pip/pip.conf

经过检查，阿里云的centos的安装源就是阿里云的源，不需要更换，所以安装的速度非常快。
